# The Old Ways — Regenerative Farming

Website for The Old Ways Farm — pasture-raised eggs available as an add-on to your regular veg box subscription.

## About

'The Old Ways' practises high-welfare farming systems which mimic nature and improve biodiversity and soil health.

Built as a single-page HTML site, deployable via GitHub Pages.

## Deploy via GitHub Pages

1. Go to **Settings → Pages**
2. Set source to **Deploy from branch → main → / (root)**
3. Site will be live at `https://sam-buddery.github.io/old-ways/`
